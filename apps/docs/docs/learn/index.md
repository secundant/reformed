## Installation

First of all, install `@reformed/core` and [effector](https://effector.dev/) (our peer dependency):

::: code-group

```sh [yarn]
yarn add @reformed/core effector
```

```sh [pnpm]
pnpm install @reformed/core effector
```

```sh [npm]
npm install @reformed/core effector
```

:::
